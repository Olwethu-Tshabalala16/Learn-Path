MASTER PROMPT (RUN FIRST — PROJECT BOOTSTRAP)

Prompt:

Create a full-stack web application named LearPath.

The app must replicate the core learning flow and structure of the Siyavula platform, but all branding must be LearPath only.

Stack (mandatory):

Frontend: HTML5, CSS3, JavaScript, React

Backend: Django + Django REST Framework

Database: SQLite

ML: Python (scikit-learn)

API: REST only

Non-negotiable rules:

Every HTML element MUST have both an id and a class

All HTML files must load ONE stylesheet: static/css/main.css

Color scheme: shining orange primary, white secondary

Frontend must use animations and transitions

Modular file structure (no monolithic files)

App must run successfully on Replit without manual fixes

Generate the full folder structure and all required files.
